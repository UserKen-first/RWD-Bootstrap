Relative Position: 相對位置 => 與物件原本的位置偏移 (其他的區塊位置不變)
Left:100px - 與原先位置距離左邊100的px => 與原先的左邊界距100px
Top:-10px - 可以使用負數

R、S : 寬度預設為整個頁面

Absoulte : 絕對位置 => 上層元件的左上角為基準點 (Body) 在整個空間內設定位置
Fixed : 以視窗為基礎 => 永遠出現在視窗的固定位置

A、F : 寬度修整為內容寬度

Float => 浮動效果，隨著視窗大小調整內容編排
解Float效果 => 直接加在想解除的物件上

可參考網站 BootStrap應用：https://getbootstrap.com/docs/4.0/content/images/
